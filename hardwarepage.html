Hardware

Motherboard

A motherboard is one of the most essential parts of a computer. It holds many components in the computer, for example, the CPU and memory. It also has connectors for input and output devices.




CPU

CPU is short for Central Processing Unit. This hardware handles all instructions it recieves from the software and hardware in the system.




RAM

RAM is short for Random Access Memory. This is the place in the computer where the operating system, application programs, and data in current use are kept so that they can be quickly used by the computer's processor.




Hard Drive

A Hard Drive is a device used to permanently store and retrieve information.




Keyboard, Mouse, and Monitor

A keyboard is a little panel of keys used to conrol an operating system. A mouse is a little device that is used to control the cursor on the monitor. The monitor is a screen that displays the information in the computer. Chances are that if you are looking at this website, you have experience with these things.
