HTML /CSS

What is HTML?

HTML is short for Hypertext Markup Language. It is a computer language devised to allow website creation. HTML consists of tags typed into a file. The image to the right shows the basic structure of an HTML page.

Here is a brief history lesson of HTML: HTML was created in the early 1990's by a man named Tim Berners-Lee. Since then, it has evolved from a simple language of with a small number of tags to a complex system of mark-up which enables authors to create web pages complete with animation and so much more.


What is CSS?

CSS is short for Cascading Style Sheet. CSS is used to change how HTML elements are displayed. It adds style to the webpage. One huge purpose CSS serves is to add color in a webpage.

How do tags work?

Tags are hidden keywords that are used to help put content onto a webpage. Typically, a tag starts with a "less than sign" and ends with a "greater than sign". Tags usually have two parts, an opening part and a closing part. Here is an example of one (no spaces are supposed to be added at the ends of the tag, this was just for demonstration):
< p >This is a paragraph tag < /p >

What are attributes?

HTML elements have attributes. Attributes provide information about an element. They are always specified in the start tag. When adding an image in a webpage, you might use the width attribute to control how wide the picture is going to be.

Lets look at some Tags

Here is a list of some tags commonly used in HTML:

  • "p"-This tag is used to create a paragraph.
  • "h1"-This tag is used to create a heading 1.(There are 6 headings h1-h6).
  • "ul" and "li"-These tags, when used together, create a list. Ul means unordered list and li is used to create the bullet points.
  • "br"-This tag is used to add breaks in between portions of text.
